# Sattvalaya Arts

A modern, fully responsive landing page designed to digitally represent a traditional Veena and holistic wellness academy. 

> **Note:** This project was developed as part of a Web Development Internship at **Techvoice**.

## 🎨 Overview
The Sattvalaya Arts website was built to transition the academy from a basic HTML presence to a sophisticated, high-performance web application. The design balances ancient tradition with modern professionalism, utilizing a dual-font system (Cormorant Garamond & Montserrat) and a warm, parchment-inspired color palette to create a premium user experience.

## 🚀 Tech Stack
* **Framework:** React, Next.js
* **Styling:** Tailwind CSS
* **Icons:** Raw inline SVGs & Lucide React
* **Deployment:** Netlify (Continuous Deployment)

## ✨ Key Features
* **Modern Architecture:** Built entirely with Next.js using component-based architecture for maximum performance and SEO capabilities.
* **Fully Responsive Design:** Utilizes CSS Grid and Flexbox to ensure perfect layouts across mobile, tablet, and desktop screens.
* **Interactive UI Elements:** Features smooth-scrolling anchor links, hover animations (`scale-105`), and a custom-built "Bento Grid" masonry gallery.
* **Direct Engagement:** Includes a persistent, floating WhatsApp deep-link button to drive direct student inquiries.
* **Performance Optimized:** Avoids heavy icon libraries in favor of lightweight inline SVGs to ensure lightning-fast load times.
