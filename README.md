# Sattvalaya Arts 🎨

This is a **Next.js** project for the Sattvalaya Arts website.

## 🚀 Getting Started

First install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

The page will auto-update as you edit the project.
